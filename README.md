## ✨ Tính năng nổi bật
- ⏰ Tự động backup vào **00:00** mỗi ngày
- 📧 Gửi email thông báo thành công/thất bại
- 🔒 An toàn với cấu hình qua file `.env`
- 📁 Tạo bản backup với timestamp tự động
